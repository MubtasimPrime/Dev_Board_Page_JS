Question-1: What are the different ways to select an element in the DOM?

There are several ways to select HTML elements. Selecting by ID, using this method we can select a single element based on its id attribute. Another one is selecting by ClassName, using this method we can select all elements that have the specified class name. Selecting with CSS selectors. One is querySelector, by this method we can have the first element that matches the specific CSS selector. Other one is querySelectorAll, it selects all elements that matches the specific CSS selector. Also selecting by Tag Name, using this we can select all elements with specific tag name.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

By using innerHTML, we can get the whole HTML part of a particular selected portion. We can have tags and text using innerHTML. On the other hand, using innerText, we can have only the visible text. And using textContent, we can get all text like if there is a span tag which is used display none, but with the help of textContent we can also see the ignores styling properties.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we can attach a single event listener to a parent element so that we can also handle events triggered on its child elements, instead of adding separate listeners to each child element. Using a single listener on a parent element, we can reduce the number of event listeners. Also it's consume less memory.

Question-4: What is event bubbling in the DOM?

Event bubbling refers to the process where an event triggered on a specific element travels up through it's parent element, then to it's grandparent, and so on, all the way up to the root of the DOM. We can say that, it's a upward flow of events through DOM. Each parent element has the opportunity to execute it's own event handler if they are defined.
